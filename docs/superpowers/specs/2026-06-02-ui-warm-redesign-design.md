# DarkTrade UI 深度优化 — 温暖陪伴 × 市场情绪

**日期:** 2026-06-02
**状态:** 设计阶段
**分支:** main

---

## 1. 设计目标

将当前功能完整但视觉风格偏"工具型"的 UI，升级为**温暖、亲和、面向新手**的体验。保持现有布局结构不变，重点重塑视觉语言。

### 核心原则

- **温暖但不幼稚** — 暖金配色 + 柔和圆角，不要萌系角色
- **市场情绪可视化** — 将 "佛系/吃瓜/上头" 情绪指标升级为跨页面的一级元素
- **新手友好** — 口语化鼓励文案、清晰的信息层级、降低认知负担
- **Web 优先** — 桌面端为主要使用场景，侧边栏导航 + 响应式适配

## 3. 布局架构 (Web 桌面端)

### 3.1 应用壳 (App Shell)

```
┌──────────┬─────────────────────────────────────┐
│          │  TopBar (标题 + 搜索)                │
│ Sidebar  ├─────────────────────────────────────┤
│          │                                     │
│  Logo    │  Sentiment Strip (市场情绪条)        │
│  Nav     ├─────────────────────────────────────┤
│  Items   │  Tab Pills (自选/持仓/热门)          │
│          ├─────────────────────────────────────┤
│  User    │                                     │
│  Card    │  Content Area                       │
│          │  (多列网格 / 双栏布局)               │
│          │                                     │
└──────────┴─────────────────────────────────────┘
```

- **侧边栏**: 固定 220px 宽，Logo + 主导航 + 用户卡片
- **顶栏**: 页面标题 + 搜索框 + 操作区
- **市场情绪条**: 横向通栏，三色标记波动程度
- **内容区**: 自适应网格 (min 320px 卡片)

### 3.2 响应式断点

| 断点 | 布局变化 |
|------|---------|
| >900px | 完整侧边栏 + 内容网格 |
| 600-900px | 侧边栏收缩为顶部导航条，只显示图标 |
| <600px | 导航移至底部固定栏 (现有 BottomNavigationBar 模式) |

### 3.3 各页面布局

**行情页**: 侧边栏 + 情绪条 + Tab切换 + 多列股票卡片网格 (2-3列)
**资产页**: 侧边栏 + 双栏布局 (左: 总额+盈亏+持仓列表, 右: 情绪卡片+资产分布)
**交易页**: 侧边栏 + 单列居中表单 (最大 480px 宽，避免表单过宽)
**个人页**: 侧边栏 + 单列 (头像+成就+VIP+菜单)

## 4. 市场情绪系统

### 4.1 判断逻辑

基于两个维度计算：

```dart
情绪 = {
  // 维度1: 市场整体波动率 (所有股票 changePct 的平均绝对值)
  平均波动 < 0.5%  → 🧘 佛系 (市场平静)
  平均波动 0.5-2%  → 🍿 吃瓜 (小幅波动)
  平均波动 > 2%    → 🔥 上头 (大幅波动)

  // 维度2: 用户交易频率 (近7天交易次数)
  交易 ≤ 1次  → 偏向佛系
  交易 2-5次  → 偏向吃瓜
  交易 > 5次  → 偏向上头
}
```

### 4.2 展示位置

| 位置 | 展示内容 |
|------|---------|
| 行情页顶部 Sentiment Strip | 市场级情绪: 图标 + 名称 + 波动率 + 一句话描述 |
| 资产页 Emotion Card | 用户级情绪: 交易频率 + 个性化提示 (如"喝杯茶🍵") |
| 行情页股票卡片 | 可选: 当某股票波动异常时加 🔥 标记 |

### 4.3 视觉表现

- 佛系: 绿色系柔光背景 + 6px 进度条偏左
- 吃瓜: 暖黄系背景 + 进度条居中
- 上头: 暖粉系背景 + 进度条偏右

| 层级 | 值 | 场景 |
|------|-----|------|

### 5.3 卡片

- 白色 `#FFFEFB` 背景 + `#ECDCC0` 1.5px 边框
- 默认微浮阴影
- hover 时边框变金色 + 阴影加强 + translateY(-2px)
- 内边距 14-18px

### 5.4 按钮

- **主按钮**: 金色实心 `#D4A853` → 白色文字 → 12px 圆角 → 金色阴影
- **次按钮**: 金色描边透明底 → 金色文字
- **买入按钮**: 绿色渐变 (仅在交易页)
- **卖出按钮**: 红色渐变 (仅在交易页)
- **Hover**: translateY(-1px) + 阴影加强

### 5.5 导航

**侧边栏导航项:**
- 默认: 透明底 + `#B8977A` 文字
- Hover: `#FDF6E8` 背景

## 6. 文案语气

### 原则

- 口语化、鼓励式，不严肃不冰冷
- 用"你"而非"用户"
- 错误提示也保持温暖

### 示例

| 场景 | 旧文案 | 新文案 |
|------|--------|--------|
| 空持仓 | 暂无持仓 | 还没有持仓哦，去探索市场吧~ |
| 空自选 | 还没有关注股票 | 去热门发现感兴趣的股票，点 ☆ 加入关注 |
| 交易成功 | 交易成功 | 成交！离大佬又近了一步 👍 |
| 余额不足 | 余额不足 | 差一点就够啦，调整一下数量？ |
| 新手提示 | (无) | 建议先用 25% 仓位小试牛刀~ |
| 高频交易 | (无) | 本周交易了 12 次，注意休息！喝杯茶 🍵 |
| 加载中 | 正在加载... | 数据正在路上... 🚀 |
| 错误重试 | 加载失败 | 网络开了小差，点此重试 🔄 |
